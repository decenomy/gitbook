## Coin Specs

• **PoW Algorithm:** X11KVS\
• **Premine:** 1,700,000 BECN \
• **PoW Blocks:** 1 - 1000\
• **PoS Blocks:** Starting from 1001\
• **Block Time:** 60 Seconds\
• **Maturity:** 100 Confirmations\
• **Prefix:** BECN adresses start with the lower case letter "b"\
• **Ports:** 36552 (p2p) / 36553 (rpc) \

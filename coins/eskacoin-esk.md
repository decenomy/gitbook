## Coin Specs

• PoW Algorithm: Quark\
• Premine: 90,000 ESK • PoW Blocks: 1 - 251\
• PoS Blocks: Starting from 252\
• Block Time: 60 Seconds\
• Maturity: 100 Confirmations\
• Prefix: ESK addresses start with the capital letter "E"\
• Ports: 14215 (p2p) / 14214 (rpc)\

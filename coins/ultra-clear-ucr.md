## Coin Specs

• PoW Algorithm: X13\
• Premine: 1,000,000,000 UCR\
• PoW Blocks: 1 - 1000\
• PoS Blocks: Starting from 1001\
• Block Time: 60 Seconds\
• Maturity: 100 Confirmations\
• Prefix: UCR addresses start with the capital letter "C"\
• Ports: 32628 (p2p) / 32627 (rpc)
